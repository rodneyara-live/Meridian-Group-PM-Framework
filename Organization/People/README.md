# People — Who's Who of the Meridian Group Ecosystem

This folder contains a profile for each relevant person in the Meridian Group, Fintrust, and Meridian Pay ecosystem. It serves as a reference directory to understand the roles, responsibilities, and work context of each team member.

---

## Purpose

- Have a quick reference of who is who, what they do, and which projects they participate in.
- Centralize work notes and personal observations to improve collaboration.
- Link people to projects via Obsidian internal links.

---

## How to use this folder

Each file represents one person. The top portion of the file (role, company, responsibilities, and projects) is filled with objective information from the One Pager or operations. The **📝 Personal Notes** section is Rodney's private space to record personal observations, work style, and a log of relevant interactions.

---

## Naming convention

```
First-Last.md
```

Examples: `David-Carver.md`, `Rodney-Ramirez.md`, `Karina-Valdez.md`
