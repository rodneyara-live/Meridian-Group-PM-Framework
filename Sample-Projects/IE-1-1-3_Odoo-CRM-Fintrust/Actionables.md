## Usage Notes

- **How to add an action item?** When generating a minute with the MINUTA prompt, Claude automatically includes the `## Action Items from This Meeting` section with the correct format. Do not add items manually here.
- **Correct format in minutes:** `- [ ] [[Responsible]]: Concrete action 📅 YYYY-MM-DD #PR-1-2-02-1 #resp/First-Last #action`
- **An item does not appear?** Verify it has the `#action` and `#PR-1-2-02-1` tags, and that the minute is saved inside `Minutes/`.
- **To close an item:** Mark the checkbox in the original minute (not here). The queries reflect the real status of the source files.
- **Required plugin:** Tasks — see [[SETUP]]
