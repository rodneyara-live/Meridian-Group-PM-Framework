# Minutes — Odoo CRM — Fintrust

Minutes of project-specific meetings, generated from transcripts using the [[Knowledge-Base/Templates/meeting-minutes/MINUTES-prompt]] prompt.

## Naming convention

```
YYYY-MM-DD-[type].md
```

Common types: `kickoff`, `seguimiento`, `tecnico`, `steering`, `cierre`, `demo`

## Note about Madrugadores

Tactical and Operational Madrugadores have their own minutes folders:
- [[Projects/Madrugador-Tactico/README]]
- [[Projects/Madrugador-Operativo/README]]

When a Madrugador minute contains relevant progress for this project, record the summary in [[../Logbook]].
