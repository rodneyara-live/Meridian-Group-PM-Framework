## 5. Lessons learned

- **QR labeling as the OpenKM access key** simplified the retrieval flow and eliminated ambiguities in case identification — replicate in other document processes.
- **Two documentation formats (Markdown + Word)** covered two different audiences without duplicating conceptual work; separate from the start who uses which format.
- **The process narration session with the DRI** (Gregory narrating, Alejandro documenting) was the most efficient method for faithfully capturing a complex operational process.
