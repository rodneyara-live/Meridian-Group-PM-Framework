## Context and objective

**Problem it solves:**
Meridian Group's file department generates significant operational costs in personnel, physical spaces and maintenance. Outsourcing its management to a specialized provider will optimize costs, free up space and improve document custody standards.

**Main objective:**
Successfully migrate Meridian Group's internal file department operations to a specialized external provider, ensuring operational continuity, document traceability and compliance with current regulations (applicable financial regulations).

**Scope:**
- Inventory and analysis of the current file department process
- Commercial proposal evaluation and SLA negotiation with the provider
- Design of the post-migration operational model
- Gradual physical migration in batches with delivery-receipt controls
- Transition of affected personnel in accordance with applicable labor law
- 3-month stabilization and monitoring period post-migration
