## Instructions for Claude — How to generate this form

> This section is for the PM / Claude. Delete when the form is complete.

**To generate the initial task proposal**, provide Claude with:
1. The **project charter** (FOR-002) or project sheet (FOR-001)
2. The **requirements matrix** (FOR-003) if available
3. **Transcripts or minutes** from discovery and planning sessions
4. The **number of milestones** agreed with the sponsor

Claude will generate a task proposal by phase, with estimated dates and suggested responsible parties. The PM reviews, adjusts, and formalizes. These tasks are the ones created in MS Planner.

**Frontmatter of this file — only 4 fields, no exceptions:**
- `type: task-plan` — document type identifier
- `project` — IE or PR project code (e.g. `"IE-1-1-3"`)
- `project_name` — readable name (e.g. `"Odoo CRM — Fintrust"`)
- `tags: [task-plan]`

> ⛔ **FOR-015 does NOT contain status, completeness, dates, or company.** Those fields live in the `Logbook.md` frontmatter. FOR-015 is exclusively a task planning document.

**To calculate overall progress:**
- Sum all task percentages and divide by the total.
- Example: 12 tasks, sum = 480 → 480/12 = 40%.
- Update `completeness` in **[[Logbook]]** (not here).

**Project dates** (baseline, planned_end, revised_end) are recorded in the `Logbook.md` frontmatter.
