# SOP-011 — Project Time Tracking Management

**Procedure for registering and controlling Upwork contractor hours in Clockify**

---

## 1. Purpose

Establish the procedure for registering and controlling hours worked by Upwork contractors on the Clockify platform, ensuring traceability and correspondence with invoiced hours.

---

## 2. Scope

This procedure applies exclusively to Upwork contractors with hourly contracts.

---

## 3. Responsibilities

- **Contractors**: Register daily in Clockify all hours worked on tasks and projects, ensuring correspondence with hours reported in Upwork.
- **Project Manager**: Monitor weekly the hours registered in Clockify. Keeps the project list in Clockify updated.
- **Upwork Administrator**: Validate that Clockify-registered hours match Upwork-invoiced hours.

---

## 4. Important Considerations

- Clockify records are for internal hour and effort tracking.
- Invoicing and payment remain exclusively through Upwork (manual hour entry or Upwork Time Tracker, at the contractor's discretion).
- **CRITICAL**: Weekly hours registered in Upwork **MUST match** the hours reflected in Clockify.
- All registered tasks MUST have a project assigned.

---

## 5. Procedure

### 5.1 Initial Registration in Clockify

1. The contractor will receive an email invitation to the Meridian Group workspace in Clockify.
2. Click "Accept" in the invitation email.
3. Create account using mandatory Meridian Group corporate email (`@meridiangroup.io`).
4. Complete the registration process.

### 5.2 Types of Recordable Activities

Hours can be registered in Clockify for:

- **Planner Tasks**: Activities planned and assigned in Microsoft Planner (main registration type).
- **Meetings**: Time spent in team, client, or project meetings.
- **Specific requirements**: Attention to specific requests (e.g., management requirements received by email).
- **Other work activities**: Any activity related to Meridian Group projects.

> **Mandatory requirement:** Every registered activity must have a project assigned.

### 5.3 Time Registration Methods

#### Method 1: Clockify Web Portal

1. Go to [https://clockify.me](https://clockify.me)
2. In the left panel, select "Time Tracker"
3. Select the corresponding project from the dropdown menu
4. Write the task description in the text field
5. Click the start button (▶) to begin registration
6. When finished, click the stop button (■)

**Correcting entries:** If you forgot to stop the tracker, you can manually edit the hours by clicking on the entry and adjusting the duration.

**Manual entry:** Select the project → indicate start and end time → add description → click "Add".

#### Method 2: Browser Extension (Recommended)

**Installation:**
1. Search for "Clockify Time Tracker" in the browser extension store (Chrome, Edge, Firefox, Safari)
2. Install the extension
3. Pin it to the toolbar
4. Log in with Clockify credentials

**Usage:**
1. Click the Clockify icon in the browser toolbar
2. Write the task description and assign the project
3. Click ▶ to start recording

**Microsoft Planner Integration:**
1. Open a task in Microsoft Planner
2. Select the task name, right-click and click the Clockify icon (it will activate automatically when detecting the text)
3. Select the corresponding project
4. Start the timer
5. When finished, stop the recording

**Resuming tasks:** Click "Continue" on a previous entry to continue recording additional time.

#### Method 3: Mobile or Desktop App (Optional)

Contractors can download the mobile or desktop Clockify app for greater flexibility in time tracking.

### 5.4 Good Recording Practices

1. **Real-time recording**: Start the timer when beginning a task and stop it when finishing.
2. **Clear descriptions**: Use specific descriptions that clearly identify the task performed.
3. **Correct project**: Always verify that the appropriate project has been selected.
4. **Daily review**: At the end of the day, review that all activities are correctly recorded.
5. **Correspondence with Upwork**: Before reporting hours in Upwork, verify they match the weekly total in Clockify.

### 5.5 Validation and Control

**Contractor weekly review:**
- Review the weekly report in Clockify
- Verify that all activities are correctly assigned to projects
- Confirm the total hours match what will be reported in Upwork

**Project Manager monitoring:**
- Weekly review of hours recorded per contractor
- Identification of inconsistencies or irregular patterns
- Communication with contractors when clarifications are needed

**Upwork Administrator validation:**
- Weekly validation of correspondence between Clockify hours and Upwork invoices
- Approval of hour reports
- Resolution of identified discrepancies

## 8. Frequently Asked Questions

**Should I register break or pause time in Clockify?**
No. Only active work time on tasks and projects is registered.

**What if I forgot to register hours from previous days?**
You can do a manual entry with the correct date from the Clockify web portal.

**Do I need to register exactly the same hours in Clockify and Upwork?**
Yes, it is mandatory that they match weekly.

**Can I register time for tasks not in Planner?**
Yes, as long as they are legitimate work activities and assigned to a valid project in Clockify.
